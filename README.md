# RDataset

RDatasets for Ruby.
* [RDatasets](https://github.com/vincentarelbundock/Rdatasets)
* [RDatasets.jl](https://github.com/johnmyleswhite/RDatasets.jl)

## Installation

```bash
gem install rdataset
```

## Usage

```ruby
require 'rdataset'
df = Daru::RDataset.new("datasets","iris")
```
